## * Branch Practical

## Create Branch

```bash
git branch branch-name
```

```bash
* git remote
* git remote -v 
```

## Switch Branch

```bash
git checkout branch-name
```

## Create File

```bash
touch branch-file.txt
```

## Add and Commit

```bash
git add .
git commit -m "Branch practical"
```

## Push Branch

```bash
git push origin branch-name
```

## Merge Branch

```bash
git checkout main
git merge branch-name
```

## Delete Branch

```bash
git branch -d branch-name
git push origin --delete branch-name
```

## Verify Branches

```bash
git branch
git branch -a
```
